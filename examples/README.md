# Examples

This directory contains example notebooks and scripts demonstrating how to use AURORA-ETC.

## Contents

- `pretraining_example.py`: Example script for self-supervised pretraining
- `finetuning_example.py`: Example script for supervised fine-tuning
- `lifelong_learning_example.py`: Example script for lifelong learning pipeline
- `drift_detection_example.py`: Example script for drift detection
- `automl_example.py`: Example script for AutoML-guided reconfiguration

## Usage

Examples will be added in future updates. For now, refer to the main scripts in the `scripts/` directory.
